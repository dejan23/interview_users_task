Interview users task

Endpoint is /users

Available query strings:

query info
/users?page= Must be a number, default is 1
/users?limit= Must be a number, default is 10
/users?search= Must be a string, search by typing email address or name
/users?sort= Must be a string, sort by 'email' or 'name'
